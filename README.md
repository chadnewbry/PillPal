# PillPal - Smart Medication Tracker

A comprehensive yet simple medication tracking app designed specifically for older adults with VoiceOver support. Features a weekly pill organizer layout (Monday–Sunday), flexible dosing schedules, and beautiful visualizations.

## Features
- **Today** — Daily pill organizer view
- **Week View** — Monday–Sunday medication schedule
- **History** — Adherence tracking and visualizations
- **Settings** — Preferences, notifications, accessibility

## Accessibility
- VoiceOver support throughout
- Large font and Dynamic Type support
- High contrast theme
- Senior-friendly UI design

## Requirements
- iOS 17.0+
- Xcode 15.4+
